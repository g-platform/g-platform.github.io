Usage

To use GCanvas, you should import GCanvas library through a <script> tag.

<script src="gcanvas-cordova-bridge.min.js"></script>
<script src="gcanvas-2d.min.js"></script>

gcanvas-cordova-bridge.min.js and gcanvas-2d.min.js locates in js/src/dist. They provides window.GCanvas object to use.

We provide two bridges, gcanvas-cordova-bridge.min.js for Android and gcanvas-webview-javascript-bridge.min.js for iOS, respectively depends on the implementation.

When useing WebGL, you need to import gcanvas-webgl.min.js instead of gcanvas-2d.min.js.

In your app, proxy a Canvas element to use GCanvas.

var canvas = document.getElementById('the-canvas');

GCanvas.proxy(canvas, function(){
  var context = canvas.getContext('2d');

  // continue your app

})

Basically, when GCanvas.proxy is called, a mask the same size with the canvas will be placed upon your page. When you draw something on the canvas, you actually draw on the mask.

  1. You should not call getContext before proxy's callback function is called.
  2. The mask will be placed at the position of the canvas when GCanvas.proxy is called. Further position changing (scroll, eg) will not change the mask's position.
  3. You can proxy only one canvas at one page.
